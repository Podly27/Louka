# Louka

Interaktivní mini hra, ve které klikáním na louku přidáváš květiny se jménem. Každá květina má 10 stupňů růstu, 11. kliknutí ji nechá vykvést a po vykvetení ji můžeš utrhnout. Květiny se ukládají do `localStorage`, takže po obnovení stránky zůstanou na místě.

## Funkce

- Přidávání květin kliknutím do louky a zadáním jména.
- Postupný růst květin a vykvetení po 11. kliknutí.
- Možnost utrhnout vykvetlou květinu dalším kliknutím.
- Kontekstové menu pro posekání louky (heslo: `unicorn`).
- Ukládání a obnova květin z `localStorage`.
